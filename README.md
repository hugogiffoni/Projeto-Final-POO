# Sistema de Gestão de Vendas - Loja de Jogos de Tabuleiro

Projeto final de "Programação Orientada a Objetos [POO]" (UAlg)

Aplicação completa para gestao de vendas de uma loja de jogo de tabuleiro, composta por:

- **Base de dados** SQLite (com normalização)
- **Web Service** em Flask (API REST)
- **Interface Gráfica** em NiceGUI (para o call center)

A GUI **nunca** comunica diretamente com a base de dados — todas as operações passam pelo web service.

---

## Tecnologias

- **Python** 3.11+
- **Flask** — Web framework
- **NiceGUI** — Interface gráfica
- **httpx** — Cliente HTTP
- **SQLite** — Base de dados
